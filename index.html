.xlcode · Python (runtime-ready)
damped oscillator 
import numpy as np

m = 1.0      # mass
k = 4.0      # spring constant
c = 0.4      # damping
dt = 0.01
t_end = 10.0

t = np.arange(0, t_end + dt, dt)
x = np.zeros_like(t)
v = np.zeros_like(t)

x[0] = 1.0   # initial position
v[0] = 0.0   # initial velocity

for i in range(1, len(t)):
    a = -(c/m)*v[i-1] - (k/m)*x[i-1]
    v[i] = v[i-1] + a * dt
    x[i] = x[i-1] + v[i] * dt
4. Upgrades
  • Parameter sweeps over c and k via .xlcompute.
  • Fit damping from experimental traces using .xlai regression.
  • Export as a live sheet with plots via .xldata and .xlflow.
